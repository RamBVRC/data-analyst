

## Description
This repository contains the solution for Task 4 of the Data Analyst Internship.
The task focuses on using SQL JOIN operations (INNER JOIN, LEFT JOIN) to analyze business data.

## Tools Used
- SQLite
- DB Browser for SQLite

## Dataset
- Chinook Database

## Files Included
- joins_queries.sql : SQL queries with comments
- joined_output.csv : Exported query results
- insights.txt : Business insights derived from analysis
- Task_4_SQL_Joins_Teja_Bellamkonda.pdf : Task documentation

## Outcome
Learned how to combine multiple tables using joins and extract meaningful business insights.
